# HiFiAdapterFilt
Convert .bam to .fastq and remove reads with remnant PacBio adapter sequences 

Dependencies:

Bamtools 
Blast+

Usage: ./pbadapterfilt.sh [ -b .bam prefix ] [ -t Number of threads for blastn. Default=1]
